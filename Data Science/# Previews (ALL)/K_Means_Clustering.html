K-Means Clustering¶

K-means algorithm identifies k number of centroids, and then allocates every data point to the nearest cluster, while keeping the centroids as small as possible.

Prepare Data¶

In [1]:
import matplotlib.pyplot as plt
import seaborn as sns; sns.set() 
from sklearn.datasets.samples_generator import make_blobs
%matplotlib inline

# create random clusters
x, y = make_blobs(n_samples=300, centers=4, cluster_std=0.60, random_state=0)
plt.scatter(x[:, 0], x[:, 1], s=50);

K-Means Clustering Classifier¶

In [2]:
from sklearn.cluster import KMeans
from sklearn import metrics

# initialize classifier
kMeans = KMeans(n_clusters=4)

# train/fit the model
kMeans.fit(x, y)

# use the model to predict the classes of the test data
y_predict = kMeans.predict(x)

# get the accuracy score
metrics.accuracy_score(y, y_predict)
