- This folder contains all the values your app is using.

- It usually contains a 'string.xml' file, 'style.xml' file, and a 'color.xml' file.

- The 'style' file is where u can configure your onw app style and theme.

- The 'string' file is where u put all ur string resources that ur app uses, it is also use to translate the app to more thatn one language.

- The 'color' file is used to store ur color resources and add new ones.

- You can always add as much resources files as you want in the values folder.
