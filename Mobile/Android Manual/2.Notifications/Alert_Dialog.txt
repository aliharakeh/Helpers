- It's a small popout window with some action buttons

- Code:

AlertDialog.Builder alert_builder = new AlertDialog.Builder(this);
alert_builder.setTitle("Alert Dialog")
             .setCancelable(false) // so that when u press on any side of the dialog screen it won't close by itself.
             .setMessage("This is an alert dialog, press any buton to close")
             // pressing any button will close the dialog automatically.
             .setPositiveButton("yes", new DialogInterface.OnClickListener() {
                   @Override
                   public void onClick(DialogInterface dialog, int which) {
                         Toast.makeText(getBaseContext(), "you pressed yes", Toast.LENGTH_SHORT).show();
                   }
               })
               .setNegativeButton("no", new DialogInterface.OnClickListener() {
                   @Override
                   public void onClick(DialogInterface dialog, int which) {
                         Toast.makeText(getBaseContext(), "you pressed no", Toast.LENGTH_SHORT).show();
                   }
               })
               .setNeutralButton("neutral", new DialogInterface.OnClickListener() {
                   @Override
                   public void onClick(DialogInterface dialog, int which) {
                         Toast.makeText(getBaseContext(), "you pressed neutral", Toast.LENGTH_SHORT).show();
                   }
               })
               .setIcon(R.drawable.ic_stat_name)
               .show();
