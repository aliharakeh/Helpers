- The intent calss can be used to use some resources from other app.

- Like using the intent class to open the browser:

      Intent i = new Intent(Intent.Action_View, Uri.parse("http://#page");
      startActivity(i);

- Or to use the Dial_pad to enter a phone number:

      Intent i = new Intent(Intent.Action_Dial, Uri.parse("tel:#number");
      startActivity(i);

- And you can use another Syntax such as:

      Inten i = new Intent(Intent.Action_View);
      i.setData(Uri.parse("http://#page");
      startActivity(i);
