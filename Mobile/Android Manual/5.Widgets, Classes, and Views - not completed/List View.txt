- It's scrollable list that contains items.

- Can contain any type of data.

- It uses the OnItemClickListener for the item select event.

- You can use a container (ex: ArrayList) to put the data in it in case you want to add new data later.

- And you can just declare an array (String[] content = {"item1","item2","item3","item4",..}).

- Code:

        final ListView listView = (ListView)findViewById(R.id.list);
        Button b = (Button)findViewById(R.id.button);
        final EditText txt = (EditText)findViewById(R.id.editText);
        final ArrayList<Integer> content = new ArrayList<Integer>(); 
        // Can be int/String/char/....etc                                                             
        // in the array adapter 3rd parameter the type is Object[]                                                            
        // so anything can be put in there.
        b.setOnClickListener(new View.OnClickListener() {
            @Override
            public void onClick(View v) {
                if(!txt.getText().toString().isEmpty())
                   content.add(Integer.parseInt(txt.getText().toString()));
                   ArrayAdapter arrayAdapter = new ArrayAdapter(getApplicationContext(),R.layout.support_simple_spinner_dropdown_item,content);
                   listView.setAdapter(arrayAdapter);
            }
        });
        listView.setOnItemClickListener(new AdapterView.OnItemClickListener() {
            @Override
            public void onItemClick(AdapterView<?> parent, View view, int position, long id) {
                // show the item's label.
                Toast.makeText(getApplicationContext(), "" + parent.getAdapter().getItem(position), Toast.LENGTH_SHORT).show();
            }
        });
