- It's an option menu that comes out according to how you set it (to a button, to a textview, etc..).

- Code:

       // define an object of the popup menu and give it the class and the view.
       PopupMenu popupMenu = new PopupMenu(MainActivity.this, view);
       // link a menu.xml to it.
       popupMenu.getMenuInflater().inflate(R.menu.popmenu, popupMenu.getMenu());
       // set the action when you click an item.
       // you can use a switch(item.getId()){..} to configure each item click event code.
       popupMenu.setOnMenuItemClickListener(new PopupMenu.OnMenuItemClickListener() {
             @Override
             public boolean onMenuItemClick(MenuItem item) {
                   // show the title/label of the item.
                   Toast.makeText(getApplicationContext(),"you choose " + item.getTitle(),Toast.LENGTH_SHORT).show();
                   return true;
             }
        });
        // to show the menu
        popupMenu.show();
